1 Solving Equation

Aim: for function $f(x)$ defined on $\mathbb{R}$, find a (one) root $x=r$ such that $f®=0$.

Def: an iterative method obeys linear convergence with rate S if $\lim_{t \rightarrow \infty}{\frac{e_{i+1}}{e_i}=S<1}$

Def: An iterative method is called locally convergent to r if the method converges to r for initial guesses sufficiently close to r.

Property:

  • Existence: find a interval $[a,b]$ for continuous $f(x)$ such that $f(a)f(b)<0$, exist a root $r \in [a,b]$.
    • Remark: r can be infinite, eg $f(x)=\begin{cases} x+1,x<-1 \ 0,x\in [-1,1] \ x-1,x>1 \end{cases}$ between -2 and 2.
  • Uniqueness: root $r$ may not be unique and mostly depends on the interval $[a,b]$ we chosen in advance.

1.1 Bisection Method

Idea: given function $f$, divide interval $[a,b]$ into two intervals $[a,c],[c,b]$ and the root $r \in[a,c]$ or $r\in[c,b]$.

  • we choose $c=\frac{a+b}{2}$

Property: Bisection method is linear convergence.

Accurate and speed:

  1. initial: $[a_0,b_0], length=b-a, error=|c-r|<\frac{b-a}{2}$.
  2. after one iteration: $[a_1,b_1], length=\frac{b-a}{2}, error=|c-r|<\frac{b-a}{2^2}$.
  3. after n iteration: $[a_n,b_n],length=\frac{b-a}{2^n}, error=|c-r|<\frac{b-a}{2^{n+1}}$

Def: A solution is correct with $p$ decimal places if the error is less than $0.5*10^{-p}$. __(idea for tolerance) __

1.2 Fixed Point Iteration

Def: The real number r is a fixed point of function $g$ is $g®=r$ (Fixed Point)

Def: let $x_0=initial,guess$, then $x_{i+1}=g(x_i),i=0,1,2,…$ (Fixed Point Iteration)

Theorem: if $g$ is continuous and $x_i$ converge to a number r, r is a fixed point of $g$

  • (proof)

idea (why we introduce fixed point): every equation $f(x)=0$ can be turned into a fixed point problem $g®=r$ in many different ways.

How to write?

  • $x=f(x)-x=g(x)$ (1.2.1)

Existence and which $g$ is better (fewer iterations)?

  • Intuitively, when $g$ is a constant function, only one iteration is needed whatever $x_0$ is. At this time $g’=0$.
  • denoted $e_i=|r-x_i|$, we can turn (1.2.1) into $e_i=ae_{i+1}, for,g(x)=x$
  • Theorem: for continuously differentiable $g$, $g®=r$ and $S=|g’®|<1$, then the FPI converges linearly with rate S to fixed points $r$ for initial guess sufficiently close $r$
    • for $g®>1$, there can be a convergence to r for some large enough initial guess
    • useful posteriori, know the root first

Property: FIP is locally convergent, it can be linear convergent sometimes.

Example: calculate $\sqrt{2}$

  1. construct $g(x)=\frac{x+\frac{2}{x}}{2}$
  2. $S=1$ at $x=\sqrt{2}$, exist and fast

Accurate and speed

  • rarely predictable beforehand
  • need tolerance and maximum number of iterations to stop
