    Object Localization

    Object Localization

    \(x,\,y\)
    • \(x\): one image
    • \(y=(p_c,b_x,b_y,b_h,,b_h,c)\)
      • \(p_c\): confidence of an object being present or not in the image
      • \((b_x,b_y)\): the coordinate of object's center
      • \(c\): class of the object being detected (1 for car, 2 for passage,...)
    Cost function
    • when \(p_c=1\), cost=...
    • when \(p_c=0\), cost=0

    determine different \(y\) can detect object and landmark

    Bounding Box Predictions

    Problem: not all cars are fitted in fixed box

    YOLO (you only look once) algorithm
    1. dividing image into smaller grids
    2. labeling each grid for training by the last method
      • for an object in more than one grid, we assign it to the grid where its midpoint is
      • \(b_x,b_y,b_h,b_w\) are Relative coordinates to the grid
    3. training the output and input by Conv

    Advantage

    • get a concrete output of \(b_x,b_y,b_h,b_w\)
    • share the same weight in grids

    Intersection Over Union

    Both for grading and evaluating your object detection algorithm.

    \(IoU=\frac{size\;of\;intersection}{size\;of\;union}\)

    • correct if IoU at least 0.5

    Summary for YOLO

    • Input image (608, 608, 3)
    • The input image goes through a CNN, resulting in a (19,19,5,85) dimensional output.
    • After flattening the last two dimensions, the output is a volume of shape (19, 19, 425):
      • Each cell in a 19x19 grid over the input image gives 425 numbers.
      • 425 = 5 x 85 because each cell contains predictions for 5 boxes, corresponding to 5 anchor boxes, as seen in lecture.
      • 85 = 5 + 80 where 5 is because \((p_c, b_x, b_y, b_h, b_w)\) has 5 numbers, and 80 is the number of classes we'd like to detect
    • You then select only few boxes based on:
      • Score-thresholding: throw away boxes that have detected a class with a score less than the threshold
      • Non-max suppression: Compute the Intersection over Union and avoid selecting overlapping boxes
    • This gives you YOLO's final output.
